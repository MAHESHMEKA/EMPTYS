# EMPTYS - Data Preprocessing Library

EMPTYS is a comprehensive data preprocessing library designed to handle various preprocessing tasks, making it easier for data scientists and machine learning engineers to prepare datasets for model training.

## Features

- Data imputation (handling missing values)
- Encoding categorical variables
- Outlier detection and handling
- Data visualization tools
- Easy-to-use functions for dataset manipulation

## Installation

You can install the `emptys` package via pip:

pip install emptys
